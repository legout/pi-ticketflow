# seed-add-a-fixer-model-to-the-metamodels-in-t

Add a dedicated **fixer** meta-model in `.tf/config/settings.json` so the fix step (IRF workflow) can run on a model tuned for patching/edits without changing the default `general` or `worker` behavior.
