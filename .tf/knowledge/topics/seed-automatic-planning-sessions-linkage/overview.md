# seed-automatic-planning-sessions-linkage

Add automatic linkage across planning artifacts by introducing a lightweight “planning session” concept: `/tf-seed` activates a session and subsequent `/tf-spike`, `/tf-plan`, and `/tf-backlog` runs automatically record relationships and cross-reference each other.
